Instalasi & Konfigurasi

clone repositori ini "https://github.com/syidikpriam/Tes-Tehnikal-PT.-FAN-Integrasi-Teknologi.git"
Masuk ke folder presensi
Lalu buka terminal dan jalankan "composer install / composer update"
Import db_inventori.sql ke database
buka terminal dan jalankan "php artisan serve" maka Laravel development server akan running dan buka pada url yang sudah diberikan, contoh "http://127.0.0.1:8000"
